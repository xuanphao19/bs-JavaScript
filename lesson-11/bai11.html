Bài 11: return Trong Hàm

  • Trong JavaScript, từ khóa return được sử dụng để KẾT THÚC VIỆC THỰC THI của một hàm và TRẢ VỀ MỘT GIÁ TRỊ từ hàm đó (cho nơi gọi hàm) .
  • Ví dụ: 
      
      function cong (a, b) {
          return a + b;
      };

      var result = cong(5, 3)
      console.log(result) // 8
  • Khi gặp từ khóa return , tất cả mã sau nó trong hàm sẽ không được thực thi.
  • Nếu một hàm không có từ khóa return hoặc không có giá trị nào được chỉ định sau return thì khi thực thi hàm sẽ trả về undefine. console.log sẽ in ra là: undefine. Ví dụ: 
      
    function cong (a, b) {
      a + b;

      return;
    // Hàm sẽ ngừng thực thi ngay lập tức.
      console.log(result) // Không thực thi.
    };

    var result = cong(15, 3)
    console.log(result) // "undefine"
